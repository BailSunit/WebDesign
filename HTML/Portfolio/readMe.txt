List of tags

header - Specify the topic of the page
span - define a subtext for the header with specialized formatting
footer - specify contact information
nav - separate out the navigation options for the entire document
table - display the experience and qualifications in a neat format
div - to specify specific sections for specialized formatting
figure - to include the details to display figures on the page
figcaption - to specify the captions for the figures and format them in a specific fashion
