1. The description for each entry is disabled in CSS according to its specified class. Clicking the icon will enable the entry within Javascript by changing its class to one that allows display. Click it again will toggle the class.
2. Add new student button is linked to a JS function that will create a new record in the existing table. The same rules as above apply.
3. Checking the checkbox for any row will populate the last column for that row with two buttons for delete and edit. We identify the parent of the parent of the input tag for the checkbox to identify the correct row to be deleted. 
4. On selecting Edit or Delete, an Alert will be activated informing the user of what is being done. 
5. Selecting any row will cause that row to change color to orange.
